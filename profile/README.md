## 🫠 우리가 여기 모인 이유

**그래서 만들었습니다.**

## 🗓️ 어떻게 진행되나

```
📅 격주 1회
⏰ 저녁 시간 (퇴근 후)
📍 온라인 기본 + 가끔 오프라인 번개
🎤 발표 30분 → 토론 30분 → 코드 리뷰 30분
🍺 그리고 비공식 뒷풀이 (선택)
```

### 운영 원칙

1. **Show, Don't Tell** 💻
   말로만 하지 말고 코드 보여주세요. 안 돌아가도 괜찮아요. 왜 안 되는지가 더 재밌음.

2. **실패 자랑 대회** 🏆
   성공 사례보다 실패 사례가 10배 유익합니다. 부끄러워하지 마세요.

3. **모르면 모른다고 하기** 🙋
   10년차여도 모르는 거 많습니다. 솔직함이 미덕.

4. **회사 기밀 금지** 🤐
   NDA는 소중합니다. 패턴만 공유하고, 구체적인 건 가슴에 묻어요.

5. **꼰대 금지** 🚫
   나이로 찍어누르지 않기. 우린 다 같은 "AI 초보" 입니다.
   (경력은 많지만요)
